Conversational BI Dashboard"Stop writing SQL. Start talking to your data."Nykaa AI is a specialized Business Intelligence (BI) tool designed for marketing teams. It allows users to upload raw campaign data (CSV/Excel) and query it using natural language. Built with a multi-agentic AI backbone, the system ensures high accuracy, data security, and instant visualization.🚀 The Technical "Edge": Agentic Self-HealingMost Text-to-SQL tools fail when the AI "hallucinates" column names. Our system solves this with a 3-Tier Agentic Pipeline:The LLM SQL Agent (Gemini 1.5 Flash): Translates user questions into SQL using dynamic schema injection.The SQL Verifier Agent (Self-Healing): Intercepts the query. If it's invalid, it fetches the real database schema, identifies the error, and forces the LLM to "self-heal" the query before execution.The Permission Guard (Security): A regex-based security layer that blocks destructive operations (like DROP or DELETE) to ensure the data remains safe.✨ Key FeaturesNatural Language to SQL: Ask "Which campaign had the best ROI?" and get a result instantly.Dynamic Data Upload: Support for CSV and Excel files with automated sanitization.Auto-Visualization: A "Chart Selector Agent" that automatically chooses between Bar, Line, or Pie charts based on the data distribution.Conversational Memory: Remembers the last 5 turns of the conversation for follow-up questions (e.g., "Now filter that by Social Media").🛠️ Tech StackComponentTechnologyFrontendVue.js 3, Vite, CoreUI (SaaS UI)BackendFastAPI (Python 3.10)AI EngineGoogle Gemini 1.5 FlashDatabaseSQLite + PandasInfrastructureWSL (Ubuntu), Git
